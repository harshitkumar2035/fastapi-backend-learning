# FastAPI Backend Learning

A backend development project built while learning FastAPI and modern Python backend concepts.

## Features Implemented

* FastAPI Setup
* Uvicorn Server
* GET APIs
* POST APIs
* Pydantic Validation
* Logging System
* Request Middleware
* CORS Configuration
* Exception Handling
* Settings Management

## Tech Stack

* Python
* FastAPI
* Uvicorn
* Pydantic
* Pydantic Settings

## Run Locally

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
uvicorn main:app --reload
```
